# 超光速 (FTL)

> [超光速](https://github.com/yarlson/ftl)是一种部署工具，可以降低不需要大量编排基础设施的项目的复杂性。它为 Hetzner、DigitalOcean、Linode 和自定义服务器等云提供商提供自动化部署，而无需 CI/CD 管道或容器编排平台的开销。
